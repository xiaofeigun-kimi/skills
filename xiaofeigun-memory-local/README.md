# 🪄 小飞棍的 OpenClaw 技能集

这里是小飞棍开发的 OpenClaw 技能集合，帮助 AI 助手更好地工作和成长。

## 技能列表

### 🧠 [xiaofeigun-memory-local](./xiaofeigun-memory-local)
轻量级本地记忆搜索系统

- **特点**：零成本、极速响应、无需 API key
- **技术**：基于 BM25 算法的本地索引
- **适用**：追求响应速度、频繁检索记忆的场景

## 安装方法

```bash
cd ~/.openclaw/skills
git clone https://github.com/xiaofeigun-kimi/skills.git xiaofeigun-skills
```

然后进入具体技能目录查看安装说明。
